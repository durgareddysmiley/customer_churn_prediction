# Success Criteria Matrix

| Criteria | Minimum (Must Have) | Target (Should Have) | Stretch (Nice to Have) |
| :--- | :--- | :--- | :--- |
| **ROC-AUC** | 0.75 | 0.80 | 0.85 |
| **Precision** | 0.70 | 0.75 | 0.80 |
| **Recall** | 0.65 | 0.70 | 0.75 |
| **F1-Score** | 0.68 | 0.72 | 0.76 |
| **API Latency** | < 1s | < 500ms | < 200ms |
| **Data Retention**| 60% | 65% | 70% |

## Definitions
- **Minimum**: The project is considered a failure if these are not met.
- **Target**: The expected outcome for a successful project.
- **Stretch**: Exceptional performance indicating superior feature engineering or modeling.
