# Project Scope

## In Scope

The following activities are included in the scope of this project:

- Customer-level churn prediction for an e-commerce business
- Feature engineering using RFM (Recency, Frequency, Monetary) analysis
- Analysis of customer purchasing behavior and product affinity
- Building and evaluating machine learning models for churn prediction
- Deploying the churn prediction model using a web application

---

## Out of Scope

The following activities are not included in the scope of this project:

- Real-time churn prediction systems
- Product recommendation engines
- Inventory management or optimization
- Integration with paid APIs or third-party enterprise systems

---

## Timeline

The total duration of the project is **8 weeks**, distributed across:
- Data understanding and cleaning
- Feature engineering
- Model development and evaluation
- Deployment and documentation

---

## Constraints

The project is developed under the following constraints:

- Only free and open-source tools are used
- No paid APIs or cloud services are included
- Deployment is done on publicly available free platforms
- The solution must be reproducible using Docker and GitHub
